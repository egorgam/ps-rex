# ps-rex

This script can help you to find all text-based and MS Office files, and apply regex to them

*syntax:*
powershell script.ps1 <regex> <ip address of data collector> <port>

Supported formats:
.txt
.log
.xml
.htm.*
.pdf
.doc.*
.rtf
.odt
.ott
.oth
.odm
.wps
.xl.*
.csv
.ods
.ots
.sxc
.stc
.wk.*
.ppt.*
.odf
.odg
.otp
.sxi
.sti
